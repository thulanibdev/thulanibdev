## 🚀 What I'm Building

### 📚 StudyFlow — EdTech Startup
> *The ultimate productivity app built specifically for students.*

Students struggle to manage their time, track deadlines, stay focused, and organise their notes all in one place. **StudyFlow solves that.**

| Feature | Description |
|---|---|
| 📅 Smart Scheduling | Automatically organises your study timetable |
| ⏱️ Study Timer | Pomodoro-style focus sessions with breaks |
| ✅ Task Manager | Subject-by-subject task lists with deadlines |
| 📝 Note Organiser | Structured note-taking per subject |
| 📊 Progress Tracker | Visual study stats and streaks |
| 🔔 Deadline Alerts | Never miss a submission or exam again |

**Status:** 🔨 Actively building | **Target Launch:** 2026

---

### 🚛 HaulBridge — Logistics Brokerage Platform
> *Connecting shippers and truckers across South Africa — faster, smarter, cheaper.*

The South African logistics industry is fragmented. Shippers struggle to find reliable trucks. Truckers struggle to find consistent loads. **HaulBridge bridges that gap.**

| Feature | Description |
|---|---|
| 🔗 Load Matching | Match available trucks with shipments instantly |
| 💰 Quote Engine | Transparent pricing between parties |
| 📍 Route Management | Optimised delivery routes across SA |
| 🤝 Broker Dashboard | Manage deals, commissions, and clients |
| ⭐ Rating System | Build trust between shippers and truckers |

**Status:** 🚀 In Development | **Role:** Founder & Freight Broker
